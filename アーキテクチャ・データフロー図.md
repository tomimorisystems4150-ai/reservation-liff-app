## 3. エンドユーザー予約フロー（LIFF）

```mermaid
sequenceDiagram
  participant U as エンドユーザー
  participant L as LINEアプリ
  participant LIFF as GitHub Pages liff.js
  participant GAS as 店舗 GAS
  participant SS as スプレッドシート
  participant CAL as カレンダー

  U->>L: リッチメニュー / LIFF URL
  L->>LIFF: liff.html?gasApiUrl=&liffId=
  LIFF->>LIFF: liff.init / liff.login
  LIFF->>LIFF: getIDToken / getProfile
  LIFF->>GAS: getInitData + idToken + lineUserId
  GAS->>GAS: verifyLineIdToken_
  GAS-->>LIFF: shopName, menus, ...
  U->>LIFF: 日時選択・確定
  LIFF->>GAS: createBooking + idToken
  GAS->>GAS: verify + 空き枠チェック
  GAS->>SS: 予約行追加
  GAS->>CAL: イベント作成
  GAS-->>LIFF: 予約ID
  LIFF-->>U: 完了表示
  U->>LIFF: カレンダーに追加
  LIFF->>LIFF: generateICS → sessionStorage
  LIFF->>L: liff.openWindow(Worker /ics)
  L->>U: Safari → 端末カレンダー追加 UI
```

**カレンダー追加**: LIFF 本番は Cloudflare Worker `GET /ics?d=` 経由（Google アカウント非依存）。GAS `downloadICS` は LIFF からは使わない。

**認証**: LINE ログイン + GAS で ID Token 検証（`lineLoginChannelId` 設定時）
